# 哥布林巢穴 - 图片资源索引

> 将所有图片按下列文件夹结构和文件名放入对应目录，后端会自动识别并服务。

---

## 文件夹结构

```
app/images/
├── common/          # 通用兜底图（任意图片加载失败时随机使用）
├── heroes/          # 传奇英雄人物头像（平铺存放）
├── legendary/       # 传说级人物（备用）
├── soldiers/        # 小兵/单位图片（平铺存放）
└── portraits/       # AI 生成的人物肖像（五位数字编号）
```

## 快速创建空文件索引（Windows PowerShell）

```powershell
# 在 app/images 目录下执行
cd app/images

# 创建英雄头像空文件
$heroes = @(
    "女王1.png","女王2.png","女王3.png",
    "团长1.png","团长21.png","团长3.png",
    "卡拉队长1.png","卡拉队长2.png","卡拉队长3.png",
    "巫女1.png","巫女2.png","巫女31.png",
    "会长1.png","会长2.png","会长3.png",
    "血港总督1.png","血港总督2.png","血港总督3.png",
    "大贤者.png","大贤者2.png","大贤者3.png",
    "维拉诺瓦城城主1.png","维拉诺瓦城城主2.png","维拉诺瓦城城主3.png",
    "星辉1.png","星辉2.png","星辉3.png",
    "女帝1.png","女帝2.png","女帝3.png",
    "精灵女王1.png","精灵女王2.png","精灵女王3.png",
    "雫1.png","雫2.png","雫3.png"
)
foreach ($f in $heroes) { New-Item -Path "heroes/$f" -ItemType File -Force }
```
